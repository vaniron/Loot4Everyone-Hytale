# Loot4Everyone

> **⚠️ Warning: Early Access**    
> The game Hytale is in early access, and so is this project! Features may be
> incomplete, unstable, or change frequently. Please be patient and understanding as development
> continues.

Loot4Everyone is a mod that instantiate loot chests for every player like lootr or Loot4Everyone minecraft mods.
Only **new** generated loot chests can be detected by the system, it's better if you use my mod in a new world!

Currently my mod don't work with :

- Double chests

Currently my mod don't support / have bugs with :

- Multiple instances

### This mod is in alpha state, bugs can occur and game updates can break the mod, use it carefully and help me fix issues if you're interested!
